# KingoR
KingoR is a Chrome extension that automatically refreshes web pages.

## Features
-Continues to run in the background even if the popup closes.
-You can set different refresh intervals for each tab.
-Enter your desired time interval to refresh the page.

Coming soon to the Chrome Web Store...
